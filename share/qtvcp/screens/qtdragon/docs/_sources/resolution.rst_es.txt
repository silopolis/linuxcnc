=================
SCREEN RESOLUTION
=================
This GUI was initially developed for a screen with 1440 x 900 resolution. It is not resizable. 
It will work in window mode on any monitor with higher resolution but not on monitors with lower resolution.
An HD version is available at https://github.com/persei802/QtDragon_hd This repository only contains the designer
file and the handler file, so the appropriate support files must be provided separately.
